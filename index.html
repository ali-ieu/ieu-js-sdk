# IEU SDK

IEU SDK, 支持第三方登录服务接入，支持社交分享功能以及 ejoysdk。

  • 安装
  • 快速指南

# 安装

支持 tree-shaking, 建议直接安装整包

# npm
npm install @ali-ieu/ieu-js-sdk
# yarn
yarn add @ali-ieu/ieu-js-sdk
